# Яндекс 2023 — Тренировочный контест — бэкенд

## Описание
Этот репозиторий содержит решения задач с тренировочного контеста Яндекса 2023 года по бэкенду. 
Для каждой задачи представлены решения на Python и Go, а также отчеты о тестировании для сравнения производительности.
В папках задач B, E есть рисунки с пояснением идеи решения. В папках задач C, D есть небольшие примечания к решениям.
